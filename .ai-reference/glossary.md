# Glossary

- Story: the user-created page containing date, text, photos, layout, optional music, and access rules.
- Public story: the shareable view under `#/story/...`.
- Preview mode: dashboard mode that renders the public story without leaving the editor.
- Plan: the commercial tier that controls editor and sharing features.
- Gratis: synthetic free plan used as the default local state and free-tier label.
- Stripe Checkout: hosted redirect flow used for paid plan purchases.
- Webhook: server-to-server event callback that syncs billing state back into the profile row.
- Story image: a single photo associated with a story and ordered by `display_order`.
- App config: key/value table used for payment and deployment settings.
- Service role: Supabase admin credential used only in Edge Functions.
- Hash route: route stored in the URL fragment, not in the path.
